# Exported Components Reference

All Vue components exported through `window.engine.components`.

---

## Available Components

| Component | Props | Description |
|-----------|-------|-------------|
| `CharacterFace` | `character` | Renders a character's face portrait |
| `CharacterDoll` | `character` | Renders a full character doll |
| `BackgroundAsset` | `asset` | Renders a background asset with positioning |
| `CustomComponentContainer` | `slot` | Renders all components registered to a slot |
